# FLaAL
Ein Projekt zur Erstellung eines LaTeX-Packages, welches eine einfache Handhabung zur Erstellung von Automaten für Formale Sprachen mitbringt.

Dieses Projekt läuft im Rahmen meiner Bachelor-Thesis.
